# 🌟 Khushi Kumari — Personal Portfolio

> A premium, dark-themed developer portfolio designed to showcase software engineering projects, academic accomplishments, and technical skills.

---

## ✨ Design Features

- 🌓 **Rich Dark Aesthetics** — Implements a modern slate-stone and terracotta accent theme with subtle neon glowing boundaries.
- 🏎️ **Infinite Scrolling Marquees** — Showcases technical skills and project cards utilizing smooth Tailwind transitions.
- 📱 **Fully Responsive Layout** — Tailored design using architectural grids that render beautifully on mobile, tablet, and desktop viewports.
- 📊 **Landing Page Screenshots** — Interactive cards showing high-fidelity mockup previews for each main project instead of generic placeholders.
- 📨 **Integrated Contact Form** — User-friendly panel with validation for direct collaboration queries.

---

## 🛠️ Portfolio Tech Stack

| Layer | Technology | Purpose |
|---|---|---|
| **Core Framework** | Next.js 14 (App Router) | High-performance React development and API orchestrations |
| **Styling** | Tailwind CSS | Modern utility-first CSS layout engine |
| **Animations** | Framer Motion | Fluid micro-interactions and scroll-triggered animations |
| **Icons** | Lucide React | Minimalist and vector-based dashboard iconography |
| **Compilation** | TypeScript | High-durability compile-time type-safety |

## 📁 Project Showcases

The portfolio features six production-grade and hackathon projects:

1. **Goonj 🎙️** *(Lead Architect)* — An AI-powered, voice-first discovery platform using RAG to decode complex government schemes across regional languages.
2. **WDC Induction Portal 🔒** *(Full Stack Engineer)* — A secure recruitment portal featuring dual-role access control for managing club evaluations.
3. **GriefBridge 🕊️** *(Backend & API Developer)* — An AI legacy support system automating administrative tasks and building interactive memory graphs.
4. **Awaaz Shield 🛡️** *(Lead Developer)* — An AI voice-verification call companion designed to screen voice prints and detect scam intent in real-time.
5. **Cortex-AI 🏆** *(Sole Creator)* — An intelligent sports companion utilizing RAG architecture to chat on Cricket, Badminton, and Chess.
6. **EcoMindAI 🌿** *(Backend Developer)* — An AI-driven sustainability planning dashboard generating community resource forecasts and carbon action blueprints.

## 🚀 Running the Project Locally

### Prerequisites
- Node.js 18+
- npm (Node Package Manager)

### 1. Install Dependencies
```bash
npm install
```
### 2. Start Development Server
```bash
npm run dev
```

Visit `http://localhost:3000` to review the portfolio!
